Posture Scan

자세 스캐닝을 시작하겠습니다!

최대한 바른 자세로 앉아 카메라를 봐주세요. 아래 순서대로 자세를 잡은 뒤 스캔을 시작하면 됩니다.

1. 허리 세우기허리를 세우고 등받이에 가볍게 기대주세요.
2. 어깨 내리기어깨 힘을 빼고 양쪽 어깨가 카메라에 보이게 해주세요.
3. 화면 정면 보기고개가 앞으로 빠지지 않게 하고 카메라를 편하게 바라봐주세요.

Good example

  • 머리와 목: 귀가 어깨 위에 가까움
  • 어깨와 등: 어깨가 말리지 않고 허리가 세워짐
  • 팔과 손목: 팔꿈치가 몸 가까이에 있고 손목이 편안함
  • 화면 위치: 화면 상단이 눈높이와 비슷하거나 살짝 아래

Bad example

  • 머리와 목: 고개가 앞으로 빠지거나 아래로 숙여짐
  • 어깨와 등: 어깨가 둥글게 말리고 상체가 구부러짐
  • 팔과 손목: 팔이 멀리 뻗거나 손목이 과하게 꺾임
  • 화면 위치: 화면이 너무 낮거나 몸이 화면에 너무 가까움
유저의 자세를 스캔 중입니다.
