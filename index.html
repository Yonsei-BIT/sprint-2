HOW TO USE

  • 1
    런어스에서 강의 페이지 열기 영상은 아직 재생하지 마세요
  • 2
    북마클릿 클릭 북마크 바의 "강의 스크립트 추출기" 클릭 → 오른쪽에 패널이 열려요
  • 3
    영상 재생 재생 버튼을 누르면 m3u8 URL이 자동으로 감지돼요
  • 4
    FFmpeg 명령어 복사 → cmd에서 실행 강의 이름과 길이 확인 후 명령어 복사, cmd에 붙여넣기
  • 5
    TurboScribe에서 텍스트 변환 turboscribe.ai에 파일 업로드 → Korean 선택 → 다운로드
