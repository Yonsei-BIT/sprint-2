FAQ
자주 묻는 질문
API 키를 매번 입력해야 하나요?
+
아니요! 최초 1회 입력 후 자동 저장돼요.
비용이 얼마나 드나요?
+
Whisper API는 분당 약 $0.006 (약 ₩8). 50분 강의 기준 약 ₩400. 처음 가입 시 무료 크레딧으로 충분해요. 각자 자신의 API 키를 쓰니까 비용도 각자 부담이에요.
"Native Host가 설치되어 있지 않아요" 오류
+
STEP 5를 다시 실행하고, 크롬을 완전히 종료(트레이 아이콘 우클릭 → 크롬 종료) 후 재시작해주세요.
Mac에서도 쓸 수 있나요?
+
현재 Windows 전용이에요. Mac 지원은 추후 업데이트 예정.
내 강의 데이터가 안전한가요?
+
오디오 파일이 OpenAI 서버로 전송되고, 변환 후 임시 파일은 자동 삭제돼요. OpenAI는 API 요청 데이터를 학습에 사용하지 않아요.
